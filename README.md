# Quiz de Sistema e Segurança

Site estático com quiz responsivo baseado nos slides da disciplina de Segurança de Sistemas.

## O que tem no projeto

- quiz com perguntas e alternativas embaralhadas
- feedback visual de acerto e erro
- barra de progresso, timer e contagem de questões restantes
- resumo dos tópicos mais importantes para revisão
- resultado final com desempenho por tema e revisão dos erros

## Estrutura

- `index.html`
- `styles.css`
- `data.js`
- `script.js`

## Publicação no GitHub Pages

1. Envie os arquivos para a branch `main`
2. No GitHub, abra `Settings > Pages`
3. Em `Build and deployment`, selecione `Deploy from a branch`
4. Escolha a branch `main` e a pasta `/ (root)`

## Conteúdo-base

O banco de questões foi montado a partir de cinco arquivos `.pptx` da disciplina e consolidado em um conjunto de revisão voltado para prova.
